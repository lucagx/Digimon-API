Chamadas API e Endpoints

Método HTTP EndPoint Faz
GET /api/digimon Retorna lista de todos Digimons
GET /api/digimon/name/:name Retorna Digimon por Nome
GET /api/digimon/level/:level Retorna lista Digimon por Nível

Campos API

Nome Descrição Tipo
name Nome Digimon String
level Nível Digimon String
img Imagen Digimon String
